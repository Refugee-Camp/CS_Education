The Gist - Question 2

Given A, B (arrays of length n) and t (an integer). Does A or Bcontain t?

```
Algorithm 2
    
1:  for i = 1 to n do
2:     if A[i] == t then
3:         Return TRUE
4:  for i = 1 to n do
5:     if B[i] == t then
6:         Return TRUE
7:  Return FALSE
```

What is the running time of this piece of code?

```
O(n)
```
